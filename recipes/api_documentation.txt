**API RECIPES**

GET Endpoints:
  - Get all recipes:
    /recipes/
  - Get recipe by id
    /recipes/<int:recipe_id>/
POST Endpoints
  - New Recipe
    /recipes/new/
    JSON:
      {"title": <string>
       "ingredients": <string>
       "instructions": <string>
       "preparation_time": <int>
       "servings": <int>
       "kcal": <int>
       "username_id": <string>}
